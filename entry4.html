I’ve been working on a mini javascript project using circles. We learned how to make circles with functions. Using variables with var, or let, I coded circles that would randomly change color, location, and z-index to make the circles appear all over the page. My function creates divs randomly positioned around the page with circles in random colors. I used Math.random to randomize the colors and location of the circles. It is a sort of random project, but it’s fun.
